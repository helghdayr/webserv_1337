🍪 Persistent Cookie Demo

This demo shows how server-side cookies persist across browser sessions.

How to Test:

  1. Click "Open Session Manager" to see persistent server-side sessions
  2. Check F12 → Application → Cookies (you'll see session_id)
  3. Delete the cookie in F12 → Application → Cookies
  4. Refresh the Session Manager - cookie will reappear!
  5. Close browser completely and reopen
  6. Visit Session Manager again - session continues!
